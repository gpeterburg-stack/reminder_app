# Reminder App

Мобильное приложение для персонализированного отслеживания приёма лекарств. Разработано на Flutter с локальным хранением данных в JSON-файлах.

## Функциональность

- Управление списком лекарств (добавление, редактирование, удаление)
- Создание напоминаний с выбором времени и дней недели
- Локальные уведомления о приёме лекарств
- Временное отключение напоминаний
- Профиль пользователя (имя, возраст, вес, хронические заболевания)
- Управление курсами лечения с гибкой настройкой длительности
- Разделение активных и завершённых курсов лечения

## Технологии

| Технология | Назначение |
|------------|------------|
| Flutter (Dart) | Кроссплатформенная разработка |
| JSON-файлы | Локальное хранение данных |
| flutter_local_notifications | Локальные уведомления |
| path_provider | Работа с файловой системой |

## Требования

- Android 8.0 (API 26) или выше
- iOS 13.0 или выше (при сборке под iOS)
